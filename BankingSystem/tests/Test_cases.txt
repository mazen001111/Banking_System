CSE333 Banking System — Manual Test Cases

HOW TO USE THIS FILE
──────────────────────────────────────────────────────────────
Each test case lists:
  - Test ID         : unique label
  - Category        : which feature is being tested
  - Steps           : what to do in the menu
  - Input           : exact values to enter
  - Expected Output : what should appear on screen
  - Pass / Fail     : fill in after running
